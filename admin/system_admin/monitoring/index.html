    System Monitoring¶

    SNMPA Setup¶

    Each node of LeoStorage, LeoGateway and LeoManager provides a built in SNMP agent which allows to connect external systems, such as Nagios and Zabbix. You can retrieve various statistics of your LeoFS.

    LeoManager¶

    Using Grafana¶

    In order to easily monitor LeoFS, we provide LeoFS' input plugin of Telegraf1. See more details on its documentation.

    If you decide to utilize Telegraf1, you need to install InfluxDB2 and Grafana3.
